Praktikum Pemrograman Web 1

Praktikum Pemrograman Web memberikan ketrampilan kepada mahasiswa dalam pembuatan situs web yang dinamis dan dapat berinteraksi dengan user. Materi meliputi HTML, CSS, MySQL, dan Visual Studio Code.

Keterangan

  1. Buatlah Class bernama Dashed untuk heading utama
  2. gunakan selector div untuk membuat box model
  3. Buatlah Class bernama underline-red untuk kalimat dengan garis bawah merah
  4. Buatlah Class bernama shadow untuk memberikan efek bayangan
